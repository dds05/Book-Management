1. 'backend' folder in this directory consists of complete backend for the application.
2. 'frontend' folder in this directory consists of complete frontend part of the application.

3. To Serve the backend project:
  Just enter the command 'npm start' in the terminal (just make sure you are inside 'backend' folder before running the command)
4. To Serve the frontend project:
 Just enter the comman 'npm start' in the terminal (just make sure you are inside 'crud-app' folder before running the command)
5. Project will be served on localhost:3000.
